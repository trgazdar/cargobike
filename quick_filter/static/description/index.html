Product Parts Finder

Product Parts Finder helps you to create product filters that are based on a non-variant attribute. Now your customers will be able to quickly find the products they need with a quick search section.

Add especially valuable for odoo eCommerce stores owners selling spare parts and other components. You can add parts finder in any page like home page, category page, etc...

You can define your Year Make Module search to search your part based on selection criteria.

List of Features

Display parts finder (with three snippet designs) on any category, Home Page and CMS pages.
Fetch drop down filters values dynamically based on parent value.
Reset child drop-down values if a parent value changed.
Search values in a drop-down if you have so many values.
Add numerous selection options to the filter.
Create as many product finders groups as needed and place on a website.
Redirect to the shop (Product Listing) on apply filter.
Add Quick Filters in your products with the new tab. It is not linked with any variant.
A quick search for desired products/parts.
