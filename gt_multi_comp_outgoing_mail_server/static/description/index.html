Different Outgoing emails for each companies setup in odoo

This module will help you to configure different outgoing emails based on (multi) companies setup in odoo

This module will provide with below features

  • Configure outgoing emails for each company
  • In outgoing emails select company for which email to use for which company

Configuring for first Company

Here you can configure outgoing emails from which you want to send your customer & Supplier emails and also select the company (company_1) for which company you want to setup this outgoing email id

Now lets create sales order, select the company or it will be autopopulated based on the current company you are

After confirming sales order we will send email to customer selected in sales order.

You can see the "From" email id which we configured for the company_1 can be verified by sales order number highlighted in red

Configuring for Second Company

Here you can configure outgoing emails from which you want to send your customer & Supplier emails and also select the company (company_2) for which company you want to setup this outgoing email id

Now lets create sales order, select the company or it will be autopopulated based on the current company you are

After confirming sales order we will send email to customer selected in sales order.
